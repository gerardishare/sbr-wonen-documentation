# Route 1: met CSV-bestand

{% stepper %}
{% step %}
### Download het sjabloon

Download het Sjabloon Aedes-benchmark (Excel). Maak daarmee het csv-bestand met onderhouds- en nieuwbouwkosten per vhe. Verzamel in het sjabloon de gegevens voor het portaal.

{% content-ref url="stap-1-download-het-sjabloon.md" %}
[stap-1-download-het-sjabloon.md](stap-1-download-het-sjabloon.md)
{% endcontent-ref %}
{% endstep %}

{% step %}
### Converteer naar XBRL

Maak via de validatieservice een XBRL aan van het csv-bestand. Download het genereerde XBRL-bestand.
{% endstep %}

{% step %}
### Importeer in het portaal

Start een rapportage in het portaal door de XBRL te importeren. Vul alle overige gegevens in in het portaal.
{% endstep %}

{% step %}
### Verzend rapportage

Voeg aan de voltooide rapportage het csv-bestand toe als bijlage. Verzend de rapportage. De aanlevering is voltooid bij status Afgerond.
{% endstep %}
{% endstepper %}
