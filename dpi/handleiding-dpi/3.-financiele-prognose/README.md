# 3. Financiële prognose

In onderdeel 3 ‘Financiële prognose’ van **dPi2025** wordt gevraagd naar:

* De balans;
* Het kasstroomoverzicht.

Afhankelijk van het gekozen scheidingsregime en de activiteiten binnen een corporatie is sprake van de volgende niveaus waarop u de **dPi2025** invult:

* DAEB TI;
* Niet-DAEB TI;
* Geconsolideerde niet-DAEB verbindingen.

Naast het onderscheid in niveaus, wordt op een aantal onderdelen ook (of in plaats hiervan) onderscheid gemaakt tussen enkelvoudige en geconsolideerde gegevens van de woningcorporatie. In [hoofdstuk Algemeen kopje ‘onderscheid regimes’](../algemeen.md#onderscheid-regimes) van deze handleiding is gespecificeerd welke hoofdstukken voor welke regimes van toepassing zijn. Verder is toegelicht welke velden in de overzichten bewust zijn geblokkeerd. De inhoudelijke toelichting op de [gegevensdefinities ](https://sbr-wonen.nl/gegevensdefinities/)staat op SBR-wonen.nl.
